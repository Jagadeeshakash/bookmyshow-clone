# 🎬 BookMyShow Clone — Full Stack MERN Application

A full-stack movie ticket booking platform built with the MERN Stack, inspired by BookMyShow. Supports real-time seat selection, Stripe payments, PDF ticket generation, and an Admin Dashboard.

---

## 🚀 Live Features

- 🎥 Browse 32 movies across 3 languages and 3 formats (2D, IMAX, 4DX)
- 💺 3 seat tiers — Premium, Gold, Silver with real-time seat locking
- 🎟️ Coupon validation and discount system
- 💳 Stripe payment integration with webhook support
- 📄 PDF ticket generation after booking
- 📧 Automated email notifications
- 🔐 JWT authentication with Role-Based Access Control
- 👨‍💼 Admin Dashboard with Chart.js graphs and KPI stats
- 🏛️ Manage 32 movies across 6 Chennai theatres
